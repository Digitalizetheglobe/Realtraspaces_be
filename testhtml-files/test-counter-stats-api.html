Counter Statistics API Test Interface

Test the Counter Stats API endpoints. The API manages counter statistics for Years in Business, Properties Sold, Happy Clients, Awards Won, and 2 demo fields.

Current Counter Statistics

1. Get Current Counter Stats

GET /api/counter-stats/

2. Get All Counter Stats

GET /api/counter-stats/all

3. Create New Counter Stats

POST /api/counter-stats/create

4. Update Active Counter Stats

PUT /api/counter-stats/active

5. Update Counter Stats by ID

PUT /api/counter-stats/:id

6. Delete Counter Stats by ID

DELETE /api/counter-stats/:id

7. Reset Counter Stats to Default

POST /api/counter-stats/reset?confirm=true

⚠️ This will deactivate all existing counter stats and create new default ones!
